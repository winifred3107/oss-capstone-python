# Open Source Software Audit: Python
**Student Name:** Sunandini
**Project:** OSS Capstone 2026

## Project Overview
This repository contains a suite of Bash scripts developed for an audit of the Python Open Source environment on a Linux (WSL) system.

### File Descriptions:
* **script1.sh**: System Identity & Kernel Audit.
* **script2.sh**: Python Package Installation Verifier.
* **script3.sh**: Directory Permission & Security Auditor.
* **script4.sh**: System Log Error Analyzer.
* **script5.sh**: Interactive OSS Philosophy Generator.

## How to Run
1. Clone the repo: `git clone [Your-URL-Here]`
2. Give permissions: `chmod +x *.sh`
3. Execute: `./script1.sh`
